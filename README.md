# AI 歌词生成器 (xagent_songsGen)

这是我接入 LLM 实现类 Agent 项目的首个尝试。

## 开发目标
- **AI 歌词生成器**：利用大型语言模型 (LLM) 根据用户提供的主题、风格或情感生成高质量的歌词。

## 功能构思
- [ ] 输入主题（如：暗恋、梦想、离别）生成歌词。
- [ ] 指定音乐风格（如：流行、摇滚、说唱）。
- [ ] 设定情感基调。
- [ ] (可选) 接入 Agent 逻辑，让 AI 能够自主优化歌词或进行多轮修改。

## 技术栈
- **Node.js**
- **OpenAI SDK**
- **LLM (如 GPT-4)**
